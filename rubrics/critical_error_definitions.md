# Critical-error definitions

| Error type | Operational definition | Maximum grade |
|---|---|---:|
| Type Misclassification | Incorrect hearing-loss type, such as SNHL classified as CHL | D |
| Severity Misclassification | Degree differs by two or more categories | D |
| Fabricated or Erroneous Test Result | A reported measurement or test outcome is absent from or contradicts the case material | D |
| Laterality Confusion | Left/right ear data are swapped or misattributed | D |
| Missed Red Flag | Failure to identify an urgent condition | D |
| Dangerous Recommendation | Clinically harmful or contraindicated advice | F |

The public numeric result file uses only these generic categories. Free-text
adjudicator rationales and case-specific error descriptions are not distributed.
